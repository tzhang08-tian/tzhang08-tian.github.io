# CrazyFood Card Library

Card data is persisted in `localStorage` under the key `crazyfood_library`.

Each saved card entry:
```json
{
  "id": 1716000000000,
  "timestamp": "2026-05-20T00:00:00.000Z",
  "imageDataUrl": "data:image/jpeg;base64,...",
  "foodName": "Vegetable Pizza",
  "calories": "308"
}
```

This folder is the designated storage location for any exported card files.
